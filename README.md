## Formação C# - Trilha Completa

Este repositório reúne o conteúdo prático desenvolvido ao longo de uma formação completa em C# e .NET, organizada em níveis de evolução do básico ao avançado.

### Nível 1 – Setup do ambiente e Fundamentos de C#
- **Objetivo**: preparar o ambiente de desenvolvimento e dominar os conceitos básicos da linguagem.
- **Conteúdos típicos**:
  - Instalação do .NET SDK e IDE (Visual Studio / VS Code).
  - Estrutura de um projeto C#.
  - Tipos primitivos, variáveis e constantes.
  - Operadores aritméticos, lógicos e relacionais.
  - Estruturas de decisão e repetição.

### Nível 2 – Fundamentos do .NET
- **Objetivo**: entender a plataforma .NET e seus recursos principais.
- **Conteúdos típicos**:
  - Conceito de CLR, CTS e BCL.
  - Estrutura de projetos e soluções.
  - Namespaces e assemblies.
  - Manipulação de coleções.
  - Tratamento de exceções.

### Nível 3 – Criando uma API com CRUD completo
- **Objetivo**: construir uma API REST em .NET com operações de CRUD.
- **Conteúdos típicos**:
  - Criação de projeto Web API.
  - Organização de controllers e endpoints.
  - Modelos (DTOs, view models e entidades).
  - Validações básicas.
  - Testes da API com ferramentas como Postman/Insomnia.

### Nível 4 – Arquitetura em camadas no .NET
- **Objetivo**: aplicar boas práticas de organização de código utilizando arquitetura em camadas.
- **Conteúdos típicos**:
  - Separação em camadas (API, Application, Domain, Infrastructure).
  - Padrão Repository e Services.
  - Mapeamento entre modelos (por exemplo, AutoMapper).
  - Inversão de dependência entre camadas.

### Nível 5 – DDD, Banco de dados, SOLID e Injeção de dependência
- **Objetivo**: aprofundar na modelagem de domínio e boas práticas de código.
- **Conteúdos típicos**:
  - Conceitos básicos de Domain-Driven Design.
  - Modelagem de entidades, agregados e value objects.
  - Integração com banco de dados (por exemplo, Entity Framework Core).
  - Princípios SOLID aplicados em C#.
  - Injeção de dependência e configuração de IoC container.

### Nível 6 – Autenticação, migrations e testes de unidade
- **Objetivo**: tornar a aplicação mais robusta, segura e testável.
- **Conteúdos típicos**:
  - Autenticação e autorização (JWT, Identity, etc.).
  - Migrations e versionamento de banco de dados.
  - Criação de testes de unidade (xUnit, NUnit ou similar).
  - Boas práticas de testes (mock, fakes, stubs).

### Nível 7 – Deploy com Azure
- **Objetivo**: publicar a aplicação em ambiente de nuvem.
- **Conteúdos típicos**:
  - Conceitos básicos de Azure para aplicações .NET.
  - Configuração de ambiente (App Service, banco de dados gerenciado, etc.).
  - Publicação da aplicação a partir da IDE ou pipeline.
  - Noções de monitoramento e logs após o deploy.

---

### Como navegar neste repositório
- **Pastas de módulos**: cada módulo/nível da formação possui sua própria pasta com projetos e exemplos.
- **Projetos de exemplo**: explore os projetos para ver a evolução dos conceitos ao longo da trilha.
- **Sugestão**: siga os níveis em ordem, pois cada etapa aproveita o conhecimento construído na anterior.
